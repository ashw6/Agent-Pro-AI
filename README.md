# Agent-Pro-AI
# 🤖 Agent Pro – AI Assistant

Agent Pro is a powerful ChatGPT-like AI assistant built using Streamlit, LangChain, and Ollama.
It supports voice input, image understanding, and PDF-based question answering.

---

## 🚀 Live Demo

👉  https://ai-student-assistant-qjbftpee4bmx2tzltexqhu.streamlit.app/

---

## 🚀 Features

* 💬 ChatGPT-style conversational interface
* 🎤 Voice input (speech-to-text using Whisper)
* 📷 Image text extraction (OCR using Tesseract)
* 📄 PDF Question Answering (RAG with FAISS)
* 🧠 Conversation memory (context-aware responses)
* 🔐 Multi-user login system
* 🗂 Chat history stored using SQLite

---

## 🛠 Tech Stack

* **Frontend:** Streamlit
* **Backend:** Python
* **LLM:** Ollama (LLaMA3)
* **Vector DB:** FAISS
* **Database:** SQLite
* **Libraries:** LangChain, Faster-Whisper, PyTesseract

## ▶️ How to Run Locally

### 1. Clone the repository

```bash
git clone https://github.com/ashw6/Agent-Pro-AI.git
cd Agent-Pro-AI
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Install Ollama

Download from: https://ollama.ai

### 4. Pull the model

```bash
ollama pull llama3:8b
```

### 5. Run the application

```bash
streamlit run app.py
```

### 6. Open in browser

```
http://localhost:8501
```

---

## ⚠️ Important Notes

* This project runs locally using Ollama (no API required)
* Ensure Ollama is running before starting the app
* Camera and voice features depend on your system permissions
